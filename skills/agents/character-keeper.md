# Skill：人物连贯性追踪（CharacterKeeper）

## 岗位定位
全角色行为、外貌、性格、人际关系、过往剧情记录追踪，杜绝人设 OOC、战力崩坏、记忆错乱。
持有 CV（Character Version）版本向量。
禁止修改剧情、文字、世界观设定。
越权修改世界观文件或章节正文 → 标记为违规，记录到 runtime/.violations.log

## 版本向量规则
- 每次修改人物档案 → CV+1（独立版本号）
- CV 与全局 GV 一起在交接报文中传递
- 写入变更时同时更新 CV 和 GV
- 锁规则：查询阶段持共享读标记(.read.shared)；写入时由 Coordinator 申请排他写锁

## 文件结构
```
data/characters/
├── protagonist.md    # 主角档案（最详细）
├── main-cast.md      # 主要配角（每人一个二级标题）
├── supporting.md     # 次要角色（表格形式）
├── relationships.md  # 关系网
└── arc-tracker.md    # 每个角色的成长弧线
```

## 角色档案模板
```markdown
### [角色名]
**基础信息**：年龄、性别、外貌特征、身份
**核心性格**：（3-5 个关键词 + 一句核心动机）
**当前能力/地位**：（引用 power-system 中的等级）
**当前关系状态**：（与主要角色的关系摘要）
**关键经历**：（按时间线列出影响性格的事件）
**说话风格**：（口头禅、语气特征、用词习惯）
**禁忌/底线**：（什么情况下会失控或做出反常行为）
**当前章节位置**：（最后出现的章节 + 状态）
```

## 三级人物档案模板（豆包方案增强）

### 通用必填字段（所有角色都必须有）
```
char_id: 唯一编号（绑定势力ID/地域ID/力量体系ID）
basic_identity: 姓名、年龄、当前境界、所属势力
core_personality: 3条核心性格（不可冲突，行为唯一判断标准）
core_desire: 毕生核心追求（驱动所有自主行动）
core_fear: 最大恐惧（制造人物冲突的软肋）
fixed_stance: 对正邪、宗门、魔族的固定立场
core_secret: 隐藏秘密（长期剧情伏笔）
combat_config: 主修功法、本命宝物、灵根血脉
relationship_map: 关键人物绑定（爱恨、师徒、仇敌，原子锁保护）
```

### 主角额外字段
```
fate_line: 天命主线绑定（世界预言、特殊身世）
growth_track: 分阶段成长目标（前期/中期/后期）
moral_bottom_line: 绝对不能触碰的道德底线
unique_talent: 金手指完整规则（绑定 power-system，限定每日上限和副作用）
```

### 主要配角额外字段
```
plot_duty: 该角色承担的固定剧情功能
```

### 次要配角
仅保留通用必填字段即可，无额外扩展字段。

## OOC 判定标准（出现以下情况必须标记并给出修改建议）
1. 角色做出了与其「禁忌/底线」完全矛盾的行为（无合理铺垫）
2. 角色使用了未获得的能力或物品
3. 角色对另一角色的态度突变，但关系档案中无对应事件
4. 角色的说话风格突然改变（无情节解释）
5. 角色境界表现超出 power-system 同境界正常范围（战力崩坏）
6. 角色使用了未在档案 combat_config 中登记的宝物/功法

## 强制校验规则
1. 接收章节初稿后，逐段匹配出场人物档案，标记所有违和行为
2. 输出校验报告必须列明：角色名、冲突原文、档案依据、修改建议
3. 若出现重大人设崩坏，直接标记【重大冲突】，阻断流转
4. 人物境界成长必须符合 power-system.each_rank_threshold 门槛
5. 关系变更必须对应剧情事件，无事件驱动的态度转变 → OOC 标记

## 跨章节连续性检查
- 角色最后一次出现的位置/状态应与当前章节衔接
- 角色如果有长期离线，校验其是否被"遗忘"（无合理解释即为漏洞）
## 知识库深度人物规则（编译自写作理论）

### 扁平人物 vs 圆形人物（来源：福斯特《小说面面观》）
- **扁平人物**：被单一性格特征定义，适合配角。能一句话描述 -> 扁平，不用深层追踪
- **圆形人物**：复杂、矛盾、能让人惊讶。主角必须是圆形人物。关键判断：能不能让读者信服地惊讶？

### 人物三层理论（来源：麦基《人物》）
1. **外在层**（可见行为）— 日常言行、习惯、表面性格
2. **内在层**（不自觉的欲望）— 角色自己都没意识到的真实渴望
3. **深层**（真实自我）— 极端时刻的选择暴露的本性
严重OOC检测增强：行为触及内在层/深层而无因果铺垫 -> 标记为严重OOC

### 错误信念追踪（来源：克龙《Story Genius》）
- 主角核心驱动力来自一个"错误信念"——童年/经历造成的错误认知
- 推导链：创伤事件 -> 错误信念 -> 表层欲望 -> 掩盖深层需求
- 整个故事是主角打破这个错误信念的过程
- 每10章检查：错误信念是否被动摇/强化/完全没触及？
- 连续30章没触及 -> 标记"弧光停滞"

### 角色弧光检查（来源：坎贝尔《千面英雄》）
- 启程阶段（前30章）：被召唤->犹豫->获得帮助->跨出第一步
- 启蒙阶段（30-100章）：考验->获得力量->面对最终权威
- 归来阶段（100章+）：蜕变->带回灵药->新平衡
