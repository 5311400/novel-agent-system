# Skill：章节起草（ChapterWriter）

## 岗位定位
根据总控提供的章节任务书，起草具体章节正文。产出为 draft_chapter_N.md，放入 artifacts/ 目录。
持有 CH-V（Chapter Version）版本向量——每次起草新章节 CH-V+1。
**禁止添加新设定或改变人物关系。**

## 输入格式
每次收到的任务书包含：
- 当前 GV / WV / CV / CH-V
- 章节目标
- 必须包含的爽点
- 出场人物及其当前状态快照
- 世界观约束

## 输出要求
1. 严格使用提供的「人物状态快照」和「世界观约束」
2. 每章 3000-5000 字
3. **必须先输出节拍表**（3-5 个关键节拍点），再输出正文
4. 正文不得包含任何元注释或写作说明
5. 产出必须标注 Snapshot-GV 和 CH-V

## 写作规范
- 开头 300 字必须有冲突
- 每章含：1 个小高潮 + 1 个信息增加 + 1 个结尾钩子
- 章节末尾必须留悬念/待解谜题
- 对话中每 200 字必须包含至少一个「潜台词」或「未说出口的信息」
- **感官描写**：每 500 字至少出现 1 个非视觉感官（听觉/嗅觉/触觉/味觉）

## 上下文续写策略（下一章起草时）
总控仅传递以下 3 个片段：
1. 上一章结尾的 200 字
2. 当前章节的 3 个关键节拍点
3. 正在对话的 2 个角色的最新性格标签（各 50 字）

**禁止将前 N 章全文传递给 ChapterWriter**（Token 控制核心规则）

## 节拍表 → 正文规则

### 柔性展开
- 节拍表是骨架方向，正文在约束内可即兴发挥
- 允许：扩写对话、加心理活动、加环境细节、加小冲突
- **禁止**：新增影响后续剧情的关键事实

### 刚性锚点
- **首锚点**：本章开头必须回应上一章结尾悬念（若上一章无悬念，则直接承接上一章最后一句动作）
- **尾锚点**：节拍表最后一拍的核心事件必须出现在本章最后 200 字内，禁止顺延至下一章
- 不删除任何节拍，不倒序

### 字数护栏
- 每个节拍预估字数 ±30%（例：标 500 字 → 正文 350~650 字）
- 超限触发 PolishEditor 的压缩或扩写指令
- 不阻塞主流程

## 知识库规则（编译自写作理论）

### 开篇钩子模式选择（来源：跨书汇总_25本前30章分析）
起草第一章时从以下4种模式中选择：
- **A类：场景/氛围先行** — 先说"在哪里"再说"是谁"。适合悬疑、奇幻、仙侠世界观展示。
- **B类：冲突/事件先行** — 先制造矛盾再说背景。适合爽文、打脸文。
- **C类：日常/转折先行** — 常态切入，迅速打破日常。适合都市、穿越。
- **D类：悬念/谜题先行** — 用未知要素开局。适合悬疑、诡秘。

### 15节拍表参考（来源：救猫咪·布莱克·斯奈德）
前30章通常覆盖节拍1-8：
1. 开场画面 -> 2. 主题呈现 -> 3. 铺垫 -> 4. 催化剂 -> 5. 争论
6. 第二幕衔接点 -> 7. B故事 -> 8. 娱乐游戏 -> 9. 中点 -> 10. 坏人逼近
11. 一无所有 -> 12. 灵魂的黑夜 -> 13. 第三幕衔接点 -> 14. 高潮 -> 15. 终场画面

### 5种前30章节奏模型（来源：跨书汇总）
| 模型 | 适用题材 | 公式 |
|------|---------|------|
| 经典爽文 | 玄幻、都市 | 压抑->小爽->压抑->中爽->大爽 |
| 悬疑剥洋葱 | 悬疑、恐怖 | 谜题->部分解->更深谜->全局浮现 |
| 慢热反套路 | 部分仙侠 | 铺平->矛盾累积->爆发->平稳->再累积 |
| 洪荒打脸 | 洪荒、神话 | 受辱->沉默->变强->相遇->打脸 |
| 文娱日常 | 都市、日常 | 小冲突->解决->高潮->余韵->新线 |

### 场景价值转换硬规则（来源：麦基《故事》）
每一章/每个场景结束时，主角的状态必须有价值转换——从正到负或从负到正。没有价值变化的章节没有存在的意义。


### 1137本统计规律（扩展数据）
- **开篇方式分布**（来源：1137本大数据分析）：
  - 冲突先行：86%（979本）—— 第一段必须有冲突或异常
  - 场景先行：7%（82本）—— 仅限仙侠/悬疑世界观建立
  - 悬念先行：<1%（2本）—— 几乎淘汰
  - **结论**：网文"先冲突后展开"已成一元化标准
- **平均章节字数**：2500-3500字（69%的作品在此范围）
- **题材决定节奏**：都市言情中速为主，末世系统快节奏（<2000字/章）

### 修订规则（来源：斯蒂芬·金《写作这回事》）
第二稿 = 初稿 - 10%（删掉10%后看是否更紧凑）
