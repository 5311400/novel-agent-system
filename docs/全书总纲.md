# 全书总纲 ·《大医长歌》

> **本总纲基于 Snapshot-GV: 0 制定，落地后 GV→1**
> 体裁：历史小说（史实为骨，合理虚构，标注边界）
> 结构：分卷式长篇，共 7 卷
> 时间跨度：东汉张仲景 → 当代国医大师
> 叙事：每卷设虚构串联人物，见证名医生平与方剂诞生
> 密度：精品深写，每卷 1-2 位名医，每位 3-5 个代表方；全书约 16 位名医、55-65 个方剂故事
> 暂定书名：**《大医长歌》**（副题：从《伤寒论》到国医大师的方剂传奇）

---

## 一、史实处理机制（四级标注）

历史小说的核心张力是"真"与"戏"的边界。本书采用四级标记，**每个方剂故事正文内以脚注/行内标记区分，章末固定附「史实注」「方义注」两栏**。

| 级别 | 含义 | 处理方式 | 举例 |
|------|------|----------|------|
| 【史实】 | 正史/医籍一手记载 | 直接采用，不改动 | 张仲景著《伤寒杂病论》；孙思邈著《千金方》；王清任赴义冢验尸 |
| 【有据】 | 见于后世医史，非一手 | 采主流说、列异说 | 张仲景"官长沙太守"（见于林亿校序等，正史无传，存争议）；华佗生卒 |
| 【推演】 | 基于已知生平的合理想象 | 允许虚构细节，符合时代与医理 | 瘟疫中行医场景、师徒辨证对话、病家情状 |
| 【虚构】 | 完全虚构 | 明确标注，不冒充史实 | 串联人物、具体病人姓名与故事、戏剧化冲突 |

**底线原则**：医理、方义、辨证、药味配伍必须准确（属【史实】级，不得虚构）；人物性格与生平大事依主流史料；凡涉及"是否真实"存争议者，一律入「史实注」列异说，不在正文下结论。

---

## 二、方剂故事化范式（四段式）

每个方剂故事固定四段结构，确保医理与文学双线推进：

```
① 缘起  —— 病家求诊 / 误诊变证 / 时代困境（戏剧冲突）
② 辨证  —— 名医识证，展示病机与医理（串联人物见证、发问）
③ 立方  —— 方剂诞生，讲清药味配伍的"为什么"（君臣佐使）
④ 转归与传承 —— 疗效兑现 + 此方如何流传后世
```

**文风配比建议**：叙事 7 / 医理 2 / 时代背景 1。医理必须"化进"情节与对话，禁止大段掉书袋。
**禁忌**：不得给方剂编造错误的君臣佐使；不得让名医说出明显违背其学派立场的医理；不得用现代词汇（如"免疫力""病毒"）出现在古代卷正文中（史实注中可用）。

## 四、分卷详纲

### 第一卷·汉魏晋《伤寒奠基》
**时间**：东汉末至西晋（建安大疫—三国—西晋）
**主线**：建安大疫中，医家从"巫祝、杂方"走向"辨证论治"体系的确立。

- **张仲景（主）**——名机，字仲景，南阳人。【有据】生卒约 150-219，正史无传，生平见于《伤寒论》原序"余宗族素多，向余二百……建安纪年以来，犹未十稔，其死亡者三分有二，伤寒十居其七"。"官至长沙太守"为后世说，存争议（入史实注）。
  - 代表方：
    1. **桂枝汤**——群方之祖，调和营卫。故事点：庸医见发热恶风猛发其汗致变证，仲景力辨"营卫不和"非表实，立和法。
    2. **麻黄汤**——发汗解表第一方。故事点：表实证"无汗而喘"，与桂枝汤对照，立"汗法"。
    3. **小柴胡汤**——和解少阳。故事点：往来寒热、正邪分争于半表半里，立"和法"另一支。
    4. **白虎汤**——清热生津。故事点：阳明经证"四大"，与承气对照（清下之别）。
    5. **四逆汤**——回阳救逆。故事点：少阴病阳衰欲脱，附子干姜炙草，生死一线。
  - 史实考据要点：建安瘟疫的病种（推测为伤寒/流行性出血热等，存疑，入史实注）；仲景是否任长沙太守；原序真伪。

- **华佗（次）**——字元化，沛国谯人。【史实】约 145-208，被曹操所杀，见《后汉书·方术传》。"刳破腹背"有载，但**麻沸散具体方已佚**，今存方皆后世伪托——本书只写"麻沸散之有无与失传"的悬念，不编造方剂组成。
  - 故事点：外科麻醉与开腹术的传说、与曹操的纠葛、医书的焚毁。

- **王叔和（次）**——西晋太医令。【史实】编次整理《伤寒论》，著《脉经》。
  - 故事点：乱世散佚中搜辑仲景遗稿，"无叔和则无伤寒论"的传承之功。

- **串联人物·卫汛**：张仲景弟子，史载有传（半史实）。以他的视角随师抄方、亲历大疫，串联各方故事，并在卷末见证叔和整理遗稿。

### 第二卷·隋唐《千金集大成》
**时间**：隋至盛唐
**主线**：医方集成时代，从"禁方秘传"走向"苍生大医"。

- **孙思邈（主）**——京兆华原人。【有据】生卒约 581-682（寿百岁余，存异说）。著《千金要方》《千金翼方》，倡"大医精诚"。
  - 代表方：
    1. **独活寄生汤**——祛风湿、补肝肾、益气血。故事点：一老将风痹多年，思邈辨"邪客痹久必虚"，祛邪与扶正并行。
    2. **温胆汤**（千金属）——理气化痰、清胆和胃。故事点：胆胃不和、虚烦不眠。
    3. **千金苇茎汤**——清肺化痰、逐瘀排脓。故事点：肺痈（肺脓肿）咳吐脓血。
    4. **犀角地黄汤**——清热解毒、凉血散瘀。故事点：热入血分、发斑吐衄。
  - 史实考据要点：生卒与年龄；"一钱散""抓药"等民间传说的真伪；《千金方》方剂归属。

- **鉴真（次）**——唐僧，688-763。【史实】六次东渡日本，带去中医药与药材。
  - 故事点：医随法渡，中医药东传的使者形象（侧重医史，方剂轻着墨）。

- **串联人物·阿苓**：虚构游方药童，随思邈采药问学，贯穿"大医精诚"精神，最终择善而从。

### 第三卷·宋金元《流派纷呈》
**时间**：北宋至元末
**主线**：中医理论高峰，金元四大家各立门户，"医之门户分于金元"。

- **钱乙（次）**——字仲阳，约 1032-1113。【史实】儿科鼻祖，著《小儿药证直诀》（弟子阎孝忠整理）。
  - **六味地黄丸**：由《金匮》肾气丸（崔氏八味丸）化裁，去桂附而成"小儿纯阳不宜温补"之方。故事点：从补肾阳到滋肾阴的范式转换。

- **李东垣（主）**——名杲，字明之，号东垣，1180-1251，真定人。【史实】师张元素，补土派，著《脾胃论》《内外伤辨惑论》。
  - 代表方：
    1. **补中益气汤**——甘温除热。故事点：战乱饥馑后"内伤发热"被误作外感，东垣辨"脾胃虚则元气衰"，立甘温除大热之法。
    2. **升阳益胃汤**——升清阳、益脾胃。
    3. **当归补血汤**——黄芪五倍于当归，"有形之血生于无形之气"。
  - 史实考据要点：金元战乱与脾胃病高发的关联；师承张元素。

- **朱丹溪（次）**——名震亨，字彦修，号丹溪，1281-1358，义乌人。【史实】师罗知悌（再传刘完素），滋阴派，著《格致余论》。"阳常有余，阴常不足"。
  - 代表方：
    1. **大补阴丸**——滋阴降火。故事点：相火妄动、阴虚火旺。
    2. **保和丸**——消食和胃。故事点：膏粱厚味致食积。
    3. **越鞠丸**——行气解郁，治"六郁"。故事点：情志致病，气郁为先。

- **串联人物·罗天益**：李东垣弟子（史实），著《卫生宝鉴》。以师承视角串联补土派与滋阴派的理路承变。

### 第四卷·明代《温补与戾气》
**时间**：明中后期至明末
**主线**：温补学派兴起，与温疫新说的萌芽交锋。

- **张景岳（主）**——名介宾，字会卿，号景岳，1563-1640，会稽人。【史实】温补派，著《景岳全书》《类经》。善用熟地，世称"张熟地"。
  - 代表方：
    1. **左归丸 / 右归丸**——真阴肾水 / 元阳命火，阴阳双补的两大范式。
    2. **熟地黄之用**——温补思路的代表性药法。
    3. **金水六君煎**等——从肾论治痰饮。
  - 故事点：从寒凉攻伐转向温补命门，与当时滥用苦寒的风气对峙。
  - 史实考据要点：温补派与寒凉派的历史论争。

- **吴又可（次）**——名有性，字又可，约 1582-1652，江苏吴江人。【史实】著《温疫论》（1642），创"戾气"说，早于西方细菌学说二百余年。
  - **达原饮**——直达膜原、辟秽化浊。故事点：崇祯末年大疫（鼠疫/瘟疫），"夫温疫之为病，非风非寒非暑非湿，乃天地间别有一种异气所感"。
  - 史实考据要点：崇祯大疫的病种推测；戾气说的超前性。

- **串联人物·沈砚**：虚构太医院医官，周旋于温补派与温疫新说之间，代表体制内医者的觉醒与挣扎。

### 第五卷·清代《温热开宗》
**时间**：清康乾至道咸
**主线**：温病学派独立成宗，卫气营血与三焦辨证双轨并立。

- **叶天士（主）**——名桂，字香岩，号香岩先生，1667-1746，吴县人。【有据】温病学派奠基，著《温热论》（门人顾景文等记录）。"先后拜师十七人"为后世传颂（入史实注，采主流说）。
  - 代表"方"与法：
    1. **卫气营血辨证法**（非成方，是辨证体系）——温热病传变层次。故事点：突破伤寒六经框架，立温病辨证新轴。
    2. **叶氏医案方化裁**（如治温热入营、透热转气诸法）。
  - 史实考据要点：叶天士无亲笔成书，著作皆门人整理；"拜师十七人"的出处与真伪。
  - **方剂归属警示**：银翘散、桑菊饮、清营汤、安宫牛黄丸**皆吴鞠通方**，本书严防误归叶天士，此点须在史实注中明示。

- **吴鞠通（次）**——名瑭，字鞠通，1758-1836，淮阴人。【史实】著《温病条辨》，创三焦辨证。
  - 代表方：
    1. **银翘散**——辛凉透表、清热解毒。
    2. **桑菊饮**——辛凉解表、宣肺止咳。
    3. **清营汤**——清营解毒、透热养阴（"入营犹可透热转气"）。
    4. **安宫牛黄丸**——清热开窍（化裁自万氏牛黄清心丸）。

- **串联人物·顾清**：虚构叶门抄方学徒，先随叶天士悟卫气营血，后见证吴鞠通承其余绪著《温病条辨》，串联两代温病传承。

### 第六卷·晚清民国《改错与汇通》
**时间**：清中后期至民国
**主线**：中医面对解剖实证与西医冲击的双重压力，自新图存。

- **王清任（主）**——字勋臣，1768-1831，玉田人。【史实】著《医林改错》。亲赴义冢义地观察刑尸、疫殓之尸四十余载，纠正古人脏腑之误。
  - 代表方（皆活血化瘀系列）：
    1. **血府逐瘀汤**——胸中血府之瘀。
    2. **补阳还五汤**——气虚血瘀、半身不遂，"气虚致瘀"的代表。
    3. **身痛逐瘀汤**——通络止痛。
    4. **通窍活血汤**——头面诸窍之瘀。
  - 故事点：以实证精神挑战千年脏腑定论，著《医林改错》"非欲后人知我，实欲后人知医"。
  - 史实考据要点：观察尸体的来源与伦理争议；其脏腑记述的得失。

- **张锡纯（次）**——字寿甫，1860-1933，盐山人。【史实】著《医学衷中参西录》，中西汇通代表。
  - 代表方：
    1. **镇肝熄风汤**——类中风、肝阳上亢。
    2. **活络效灵丹**——气血凝滞、癥瘕。
    3. **升陷汤**——胸中大气下陷（"大气"理论的发挥）。
  - 故事点：以西理解释中药、以中药印证西理的汇通实践。

- **串联人物**：虚构医校学生，先随王清任见证解剖改错之勇，后随张锡纯经历中西医论争，串联两代"求真"精神。

### 第七卷·当代《薪火相传》
**时间**：1949 至今
**主线**：中医在现代社会体制下的传承、振兴与走向世界。
**史实处理特别说明**：本卷涉及近现代/在世人物，凡方剂医案以公开出版物/国医大师审定资料为准，涉及在世者一律以公开资料为据，不做私密化虚构。

- **邓铁涛（主）**——1916-2019，广东人。【史实】国医大师，重症肌无力、冠心病。著《邓铁涛医集》。
  - 代表方/法：以医案故事化呈现其治重症肌无力、冠心病的辨证用药思路。

- **朱良春（次）**——1917-2015，南通人。【史实】国医大师，虫类药大家。益肾蠲痹丸。
  - 代表方/法：虫类药入络搜剔的用药特色。

- **串联人物·林晚**：虚构中医学院学生，求学于国医大师门下，见证中医在现代教育体制与临床一线的传承，收束全书"大医长歌"主题。

## 五、串联人物体系

| 一 | 卫汛 | 半史实（仲景弟子有载） | 随师抄方，亲历大疫 |

## 七、写作规范与禁忌

1. **医理零硬伤**：君臣佐使、辨证、病机必须准确，属【史实】级，不得为戏剧性而改动。

3. **时代词汇管控**：古代卷正文禁出现"病毒/免疫/细菌/基因"等现代词；"戾气""疫气"等当时概念可用。
4. **在世人物红线**：第七卷以公开资料为据，不虚构私密情节。
5. **每章末固定两栏**：「史实注」（四级标记出处与异说）+「方义注」（方剂组成、功效、配伍要点）。
6. **文风配比**：叙事 7 / 医理 2 / 时代背景 1，医理化入情节，禁掉书袋。
