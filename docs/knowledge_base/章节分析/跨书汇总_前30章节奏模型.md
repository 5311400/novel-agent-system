# 跨书汇总 2/3：前30章节奏模型

> 基于 25 本小说前30章分析
> 关注：**每6章为一个节奏单元，5个单元构成完整前30章**

---

## 一、五种节奏模型

### 模型1：经典爽文节奏（6本）
**代表**：斗破苍穹、一剑独尊、龙王出狱、我洪荒准圣归来、社畜逆袭、无敌六皇子  
**核心公式**：`压抑 → 小爽 → 压抑 → 中爽 → 短压抑 → 大爽`

```
章节: 1-6    7-12   13-18   19-24   25-30
情绪: 📉⬇️    📈↗️   📉⬇️    📈⬆️    📉↗️📈⬆️
冲突: 低谷    小反转  新困境  中反转  准备大高潮
```

**关键数字**：
- 爽点间隔：最大不超过5章（超过读者会弃书）
- 压抑期：3-7章（不能超过7章）
- 大爽点：每6-8章一次（前30章应有3-5个大爽点）

---

### 模型2：悬疑剥洋葱式（3本）
**代表**：十日终焉、诡秘之主、我不是戏神  
**核心公式**：`谜题 → 部分解答 → 更深谜题 → 部分解答 → 全局谜题浮现`

```
章节: 1-10    11-20     21-30
情绪: 🤔❓     🤯❗️    😱🤯
谜题: 建立    第一层揭穿 第二层浮现
```

**关键数字**：
- 每10章完成一次"谜→答→新谜"循环
- 每一层必须给出足够的"小解答"让读者获得即时满足感
- 第30章不解决全局谜题，只揭示"还有更多"

---

### 模型3：慢热成长式（4本）
**代表**：凡人修仙传、完美世界、遮天、逍遥四公子  
**核心公式**：`介绍世界 → 加入事件 → 初步成长 → 遭遇更大世界 → 准备出发`

```
章节: 1-6      7-12       13-18      19-24       25-30
情绪: 🤔平静    🌟发现     💪第一次  🌍看到更大  🏃准备出发
阶段: 日常生活  接触超凡   初获奇遇   世界观扩展开  离开新手村
```

**关键差异**：
这种模型在前30章**不追求大爽点**。凡人修仙传前30章主角还在打基础，遮天前30章主角刚刚离开地球。

**注意事项**：
- 只有极其成熟的作者能驾驭（凡人、遮天都是多年积累）
- 新手不要轻易尝试——读者可能在压抑期就弃书了

---

### 模型4：案件交替式（2本）
**代表**：大奉打更人、低智商犯罪  
**核心公式**：`日常建立 → 案件1解决 → 日常喘息 → 案件2浮现 → 案件2高潮`

```
章节: 1-6      7-12       13-18      19-24      25-30
情绪: 😊轻松    🕵️破案    😊回归    🔥危机     🕵️高潮
事件: 世界建立  第一案件   日常穿插   第二案浮   第二案解决
```

**关键特征**：
- 每5-8章完成一个独立单元（案件/事件）
- 案件之间用轻松日常做缓冲（舒缓读者情绪）
- 适合"单元剧+主线"结构的书

---

### 模型5：沙雕/反常理式（6本）
**代表**：时停时停×7、癫都癫癫点好啊、精神病面前鬼东西算个球、游戏入侵、诸神愚戏、黑月光冠冕  
**核心公式**：`反常设定 → 适应世界 → 搞笑展开 → 危机 → 继续搞笑`

这类书的结构最自由。爽点来自"反常规操作"而非"打脸升级"，节奏更依赖作者的创意密度。

---

## 二、前30章的不同写法占比

| 模型 | 占比 | 覆盖题材 |
|------|------|---------|
| 经典爽文 | 6本(24%) | 玄幻、修仙、都市、历史 |
| 沙雕反常理 | 6本(24%) | 搞笑、女频、脑洞 |
| 慢热成长 | 4本(16%) | 传统玄幻、史诗向 |
| 悬疑剥洋葱 | 3本(12%) | 悬疑、脑洞、克苏鲁 |
| 案件交替 | 2本(8%) | 推理、探案 |
| 其他/混合 | 4本(16%) | 混合型 |

---

## 三、Agent 可复用的"节奏选择器"

```
当用户说"我想写XX题材的小说"：
  if 爽文向：推荐 模型1（经典爽文节奏）
  if 悬疑向：推荐 模型2（剥洋葱式）
  if 传统向：推荐 模型3（慢热成长）—— 但需提示风险
  if 单元剧：推荐 模型4（案件交替式）
  if 搞笑向：推荐 模型5（沙雕式）
  
通用安全选择：模型1（经典爽文节奏）
  理由：25本中24%采用此模型，且容错率最高
```

## 四、所有模型的共通原则

无论哪种模型，前30章都必须完成的任务：

| 序号 | 必做任务 | 截至章节 |
|------|---------|---------|
| 1 | 读者知道主角是谁 | 前3章 |
| 2 | 读者知道这个世界有什么不同 | 前5章 |
| 3 | 第一个冲突/事件结束 | 前8章 |
| 4 | 读者知道主角要干什么 | 前15章 |
| 5 | 让读者感到"有看头" | 前30章结束时 |

这些是所有25本无一例外的共同特征。
