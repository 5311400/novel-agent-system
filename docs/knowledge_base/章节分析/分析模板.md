# 章节结构化分析模板

> 用途：把小说原文丢给我，我按此模板逐章输出结构化分析
> 输出位置：`知识库/章节分析/》

## 每章输出格式

### 【基本信息】
- 小说名：
- 章号：
- 字数：
- 节奏标签：快 / 中 / 慢

### 【节拍表】
| 节拍 | 位置 | 内容概要 | 字数 |
|------|------|---------|------|
| 钩子 | 开头 | 一句话抓住读者 | ~200 |
| 铺垫 | ... | 交代现状和背景 | ... |
| 冲突触发 | ... | 矛盾爆发点 | ... |
| 冲突推进 | ... | 你来我往 | ... |
| 高潮/反转 | ... | 关键转折 | ... |
| 余韵/钩子 | 结尾 | 让读者想看下一章 | ~100 |

### 【钩子分析】
- 开篇钩子类型（废柴开局 / 穿越 / 重生 / 系统 / 悬念 / 倒叙...）
- 结尾钩子类型（悬念断章 / 预告断章 / 反转断章...）

### 【冲突模式】
- 冲突类型（尊严 / 生存 / 利益 / 情感...）
- 对抗双方
- 冲突强度（1-5）

### 【爽点设计】
- 核心爽点模型（碾压打脸 / 扮猪吃虎 / 逆袭翻盘 / 获宝捡漏...）
- 情绪曲线：➡️ 压抑段 → 🔥 释放段 → 🎯 留钩子

### 【文风特征】
- 句式：短句为主 / 长句描写 / 长短交错
- 对话占比：~%
- 描写密度：简笔 / 中等 / 细腻

### 【可复用公式】
本章最核心的写作公式，一句话提炼。
