# 章节分析：凡人修仙传（前30章）
> 经典·慢热修仙 · 忘语
> 前30章完整（文件含32章正文）
## 风格定位
反斗破的写法——主角是山村普通少年，没有天才光环，没有复仇誓言。
斗破是"从天之骄子跌落"→逆袭；凡人是"从蝼蚁开始"→缓慢爬升。

## 关键差异 vs 斗破
| 维度 | 斗破 | 凡人 |
|------|------|------|
| 主角起点 | 曾经的天才 | 普通村民 |
| 冲突强度 | 第一章就爆发 | 前5章都在铺垫日常 |
| 金手指 | 第9章药老现身 | 更隐晦的机缘 |
| 节奏 | 紧凑 | 缓慢，细节丰富 |
| 适合学习 | 快节奏爽文公式 | 氛围和世界感营造 |
