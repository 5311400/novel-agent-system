# 小说课Ⅰ：折磨读者的秘密（许荣哲）

> 一句话：用具体案例讲具体技法，最实用的中文写作课。

## 核心章节
1. 人物 — 曹操"宁教我负天下人"一句话立住人设
2. 场景 — 史铁生《命若琴弦》的场景设计
3. 对话 — 水面上的表面对白 + 水面下的真实意图
4. 开场 — "全世界最好的小说开头"案例
5. 性格 — 性格决定命运
6. 故事 — 偷矛盾与两难的贼
7. 想象力 — 陌生化手法

## 对小说Agent的价值
中文语境下最实用的创作技法书。作者用经典小说案例讲解具体技巧，可以直接提炼成Agent的Skill：
- "如何用一句话立住人设"
- "如何写有潜台词的对话"
- "如何设计两难场景"
