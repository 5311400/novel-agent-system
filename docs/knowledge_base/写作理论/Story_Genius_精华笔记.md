# Story Genius（丽莎·克龙）

> 一句话：从认知科学出发，告诉你为什么大多数故事失败——因为你没有从角色的内在动机开始。

## 核心理念：故事从"内部"开始

传统写作教你先有情节再有角色。克龙说：**反了**。

### 第一步：找到主角的"错误信念"
- 主角内心深处有一个"自以为正确但实际上在伤害自己"的信念
- 这个信念来自于童年的某个创伤/误解
- 整个故事就是主角打破这个错误信念的过程

### 第二步：从错误信念推导欲望
- 错误信念 → 表层欲望（主角以为自己想要的） → 深层需求（主角真正需要的）
- 情节=外部事件推着主角从表层欲望转向深层需求

### 第三步：故事脑的生物学基础
- 人类大脑天生是"故事处理器"
- 读者代入主角的前提：你让我理解主角的动机
- 没有内部动机的冲突=空洞的打斗

## 核心写作流程

1. 写"主角怎么了"（What）之前，先搞清楚"为什么"（Why）
2. 每个场景的冲突必须和主角的内部斗争相关
3. 修改时必须回头看：这个场景有没有推进主角的内部转变

## 对小说Agent的价值

- **最深层的角色塑造工具**：比麦基的人物三层更重实操
- **CharacterKeeper的进化方向**：不仅追踪外部状态，还要追踪"错误信念"的变化
- **PolishEditor的深层检查**：每章的冲突是否连接到主角的内部动机
