# Writing the Breakout Novel Workbook（唐纳德·马斯）

> 一句话：不是教你"写小说"，是教你"从好到爆"——突破性畅销小说的修炼手册。

## 核心理念

本书是《Writing the Breakout Novel》的工作手册，强调"练习比理论更重要"。

### 突破性小说的三大支柱
1. **格局**（Scope）— 故事的格局决定天花板。从"救一个人"到"救一个世界"
2. **深度**（Depth）— 角色不是标签集合，是有矛盾、有秘密的人
3. **张力**（Tension）— 每一页都要有让读者担心的东西

### 核心练习领域

| 领域 | 练习内容 |
|------|---------|
| 前提升级 | 把你的故事前提改三版：每版放大格局 |
| 角色深层动机 | 给每个主要角色一个"不能说的秘密" |
| 对手升级 | 反派必须和主角一样聪明 |
| 多线情节 | 至少3条剧情线交织 |
| 设定即冲突 | 每个设定元素都应该制造冲突 |
| 情感筹码 | 读者关心的不是"会不会死"，是"主角因此失去什么" |

## 对小说Agent的价值

- **C级错误检测**：当冲突不够强时，可以用"格局升级"方法推演
- **PolishEditor的"爆款检测"**：不仅是语法检查，更是潜力评估
- **Conflict Planner的工具**：从"小品级冲突"到"史诗级冲突"的升级路径
