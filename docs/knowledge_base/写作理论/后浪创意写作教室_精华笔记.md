# 后浪创意写作教室（套装共6册）

> 一句话：六本实用写作书的合集，覆盖从入门到出书的完整路径。

## 套装内容

| 书名 | 作者 | 核心价值 |
|------|------|---------|
| 《小说的八百万种写法》 | 劳伦斯·布洛克 | 侦探小说大师的60年创作心得 |
| 《畅销作家写作全技巧》 | 大泽在昌 | 日本直木奖作家的写作讲座实录 |
| 《写小说最重要的十件事》 | 厄休拉·勒古恩 | 已有单本精华笔记 |
| 《小说的骨架：好提纲成就好故事》 | 凯蒂·维兰德 | 如何做小说提纲和笔记 |
| 《小说创作技能》 | 许荣哲 | 与小说课系列互补的实战技能 |
| 《故事思维》 | 待确认 | 故事思维在写作中的应用 |

## 核心价值

这是一个"阶梯式"的写作训练包：
1. **布洛克**讲最底层的"写出来"——不要怕烂，烂的总比没有好
2. **大泽在昌**讲职业化的"每天写"——日本作家的自律体系
3. **勒古恩**讲语言的"写好它"——用词用句的艺术
4. **维兰德**讲组织的"别跑偏"——提纲和大纲
5. **许荣哲**讲技巧的"怎么妙"——具体缝合术
6. **故事思维**讲读者的"怎么听"——叙事即说服

## 对小说Agent的价值

- 提供了完整的Writer成长阶梯：卡在哪一层就用哪一层的方法
- 维兰德的提纲方法论可转化为ChapterWriter的节拍表生成规则
- 大泽在昌的"职业作家习惯"可作为写作效率模块参考
- 作为"进阶书单"存在，每个Agent在遇到特定问题时可以查阅对应的书
