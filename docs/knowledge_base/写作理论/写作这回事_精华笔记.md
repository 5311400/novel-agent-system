# 写作这回事（斯蒂芬·金）

> 一句话：一半是回忆录，一半是写作指南——金先生教你"认真对待写作"。

## 结构

### 前半部分：简历（C.V.）
金自己的成长历程：
- 童年→写作萌芽→退稿（墙上钉了整墙退稿信）
- 《魔女嘉莉》的诞生始末（妻子从废纸篓里救出手稿）
- 成名后的写作习惯

### 后半部分：工具箱（Toolbox）
**第一层工具**（最底层，最重要的）：
1. **词汇**（Vocabulary）— 用你第一反应想到的词，不要刻意"高级"
2. **语法**（Grammar）— 主谓宾结构是骨骼

**第二层工具**：
3. **段落**（Paragraphs）— 段落是思考的基本单位
4. **句式**（Sentences）— 长短交错创造节奏

**第三层工具**：
5. **叙事**（Narrative）— 场景→人物→对话→节奏
6. **描写**（Description）— 用细节，不要用量
7. **对话**（Dialogue）— 真实对话从来不"说明"，是"暗示"
8. **主题**（Theme）— 故事写完了，主题自然浮现

## 核心信条

- **关上门写，打开门改**（Write with the door closed, rewrite with the door open）
- **每天写2000字**（职业化的第一个标志）
- **杀掉你的挚爱**（Kill your darlings）— 删掉最好的句子，如果它不服务于故事
- **初稿是"纯粹的故事"，二稿才是"展示"**
- **多读多写**（If you don'\''t have time to read, you don'\''t have the time to write）

## 对小说Agent的价值

- "第二稿=初稿-10%"可作为自动删减规则
- 金的"工具箱"分层可以映射到StyleGuard的检查层级
- 最鼓舞人心的写作指导——不是教，是唤醒
