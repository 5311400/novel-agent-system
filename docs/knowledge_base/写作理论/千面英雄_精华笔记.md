# 千面英雄（约瑟夫·坎贝尔）

> 一句话：所有"废柴逆袭"故事的母模板——英雄之旅。

## 核心框架：启程→启蒙→归来
**启程**：
1. 历险的召唤 — 主角被叫醒
2. 拒绝召唤 — 主角犹豫
3. 超自然的援助 — 金手指/导师登场
4. 跨越第一个阈限 — 走出新手村
5. 鲸鱼之腹 — 彻底告别旧世界

**启蒙**：
6. 考验之路 — 一系列挑战
7. 遇到女神 — 获得帮助
8. 妖妇的诱惑 — 诱惑/陷阱
9. 与天父和好 — 面对最终权威
10. 奉若神明 — 获得终极力量

**归来**：
11. 拒绝归来
12. 借助外力逃脱
13. 穿越归来的阈限
14. 带回灵药

## 对小说Agent的价值
这棵树是所有爽文故事的根。斗破、凡人、遮天、完美世界——25本里的绝大部分玄幻小说都遵循这个模型。可以把它作为故事结构的"校验模板"。
