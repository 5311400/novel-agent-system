# 小说面面观（E.M. 福斯特）

> 一句话：1927年的剑桥演讲，却定义了后世所有小说理论——"扁平人物vs圆形人物"即出自本书。

## 七个面向

福斯特将小说拆解为七个角度：

### 1. 故事（Story）
- 故事是最低级的审美体验——"然后呢？然后呢？"
- 但它是小说的脊椎，没有故事就没有一切

### 2. 人物（People）
- **扁平人物**（Flat Character）— 被单一性格特征定义，易认易记，适合配角
- **圆形人物**（Round Character）— 复杂、矛盾、能让人惊讶，适合主角
- 关键判断：能不能让读者信服地惊讶

### 3. 情节（Plot）
- 故事≠情节。故事是"国王死了，王后也死了"，情节是"国王死了，王后因悲伤而死"
- 情节的核心：因果关系

### 4. 幻想（Fantasy）
- 小说允许一切——但必须建立内在逻辑
- 幻想小说的最高境界：让读者相信"这个规则下，一切都是真的"

### 5. 预言（Prophecy）
- 小说中可以存在高于现实的"神性视角"
- 托尔斯泰、陀思妥耶夫斯基的作品中常见

### 6. 模式（Pattern）
- 全书的结构节奏——对称、重复、回环
- 好模式让读者在读完后仍回味

### 7. 节奏（Rhythm）
- 贯穿全书的反复出现的意象/短语——福斯特叫它"噱头"（catchphrase）
- 每次出现都有新的含义

## 对小说Agent的价值

- **"扁平vs圆形"分类**可直接用于CharacterKeeper的OOC检测
- **故事≠情节**的区分是PolishEditor检查"是否只是流水账"的依据
- **节奏模式**可转化为跨书汇总中的"节奏模型"理论基础
