# 救猫咪合集（布莱克·斯奈德）

> 一句话：好莱坞编剧的15节拍表，被网文作者广泛借鉴。

## 15节拍表（核心产出）
1. **开场画面** — 主角日常状态的快照
2. **主题呈现** — 有人对主角说"你需要XX"
3. **铺垫** — 建立世界和人物
4. **催化剂** — 事件打破日常（第1章第1个冲突）
5. **争论** — 主角犹豫要不要行动
6. **第二幕衔接点** — 主角决定行动
7. **B故事** — 爱情/友情/副线
8. **娱乐游戏** — 故事"有趣"的部分
9. **中点** — 假胜利或假失败
10. **坏人逼近** — 反派变强/压力增大
11. **一无所有** — 主角跌入谷底
12. **灵魂的黑夜** — 绝望时刻
13. **第三幕衔接点** — 找到解决方案
14. **高潮** — 最终对决
15. **终场画面** — 和开场呼应的新状态

## 对网文创作的适用
前30章恰好覆盖节拍1-8左右。节拍表可以作为你检查章节节奏的参考坐标。
