# 罗伯特·麦基虚构艺术三部曲（故事·对白·人物）

> 一句话：故事创作的理论顶峰，三部书分别讲结构、对话、角色设计。

## 《故事》核心
- **结构图谱**：故事不是靠灵感，是靠结构
- **五部分结构**：激励事件→进展纠葛→危机→高潮→结局
- **故事材质**：社会/人物/结构/意义四个维度
- **场景设计**：每一个场景都必须有价值转换（从正到负或负到正）

## 《对白》核心
- 对白不是日常对话，是"带有意图的行动"
- 对白的三层功能：信息传递、人物刻画、情节推进

## 《人物》核心
- 人物分三层：外在（可见行为）→ 内在（不自觉的欲望）→ 深层（真实自我）
- 人物弧光：人物必须变化

## 对小说Agent的价值
麦基的书是所有创作理论的源头。可以把它提炼成"场景价值检查器"：
- 每章读完判断：正→负 or 负→正？如果没有变化，这章就没用。
