## 如何与章节分析联动

```
写作理论（宏观范式）    章节分析（微观验证）
     ↓                         ↓
救猫咪15节拍表      ←→  跨书汇总_前30章节奏模型
千面英雄之旅         ←→  跨书汇总_开篇钩子模式
冲突与悬念LOCK       ←→  跨书汇总_冲突类型与爽点
麦基场景价值转换      ←→  单本分析中的节拍表
```

> 理论提供"应该怎么做"，分析提供"实际上怎么写"——两者对照就是Agent的写作决策依据。
