# 冲突与悬念：小说创作的要素

> 一句话：冲突是引擎，悬念是变速箱。

## 核心框架：LOCK 系统
- **L**ead（值得追随的主角）— 正面/反面/非英雄三种选择
- **O**bjective（生死攸关的目标）— 主角一定要达成的事
- **C**onfrontation（正面对抗）— 冲突持续不断
- **K**nockout（精彩的结尾）— 让读者觉得值

## 核心结论
没有冲突就没有戏剧性。冲突是两个以上势不两立的方面之间的意志之战。悬念是让读者"翻页"的动力，源自冲突产生的"后来怎么样了"的疑问。

## 对小说Agent的实际价值
- 提供了检查章节是否"有戏"的标准：每章必须至少有一个冲突点
- LOCK系统可用来评估一个故事设定是否稳固
- 冲突密度建议：每5-8章一个大冲突，每章至少1个小冲突
