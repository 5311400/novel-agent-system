# Story Engineering（拉里·布鲁克斯）

> 一句话：小说创作的"六大核心能力"——把写作当作工程来理解。

## 六大核心能力（Six Core Competencies）

1. **前提**（Premise）— 故事的一句话概括，必须有冲突暗示
2. **主角**（Protagonist）— 必须有欲望+缺陷+弧光
3. **冲突**（Conflict）— 欲望的障碍，必须持续升级
4. **结构**（Structure）— 四部分结构：设定/回应→上升行动→危机→高潮结局
5. **场景执行**（Scene Execution）— 每个场景必须有入点/冲突/出点
6. **文笔**（Writing Voice）— 风格是最后的锦上添花

## 四部分结构（替代三幕剧）

| 部分 | 占比 | 功能 |
|------|:----:|------|
| 第一部分：设定与回应 | ~25% | 建立世界、主角困境，引出核心问题 |
| 第二部分：上升行动 | ~25% | 主角尝试解决问题，不断失败 |
| 第三部分：危机 | ~25% | 主角面对最大的挑战，似乎要输了 |
| 第四部分：高潮与结局 | ~25% | 最终对决，解决问题 |

## 对小说Agent的价值

- **前提审查**：每章写完检查是否偏离前提
- **结构定位**：第N章应该处于四部分的哪个位置
- **场景检查**：每个场景是否有入→冲突→出
- **最直接可工程化的写作书**：每项能力都可量化检查
