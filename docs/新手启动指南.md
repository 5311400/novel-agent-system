# 新手启动指南 — 从零到第一章

## 第一步：初始化系统

首次使用，执行：

```
/init
```

Coordinator 会引导你填写 3 项核心设定：
1. **故事一句话梗概**（含核心冲突）
2. **主角名 + 核心性格（≤5 个词）**
3. **世界观唯一核心设定**（如"唯一天道已死，灵力溃散"）

## 第二步：填写档案（可选）

系统自动生成 data/world/ 和 data/characters/ 的模板。
你也可以在 data/world/*.example 中查看示例数据（《青云大世界》），
然后按格式修改对应的 .md 文件。

档案填写顺序建议：
1. power-system.md → 先定力量体系（最大陷阱）
2. geography.md → 地理地图
3. factions.md → 势力宗门
4. rules.md → 世界底层规则
5. protagonist.md → 主角档案
6. main-cast.md → 主要配角

## 第三步：开始写作

档案就绪后，执行：

```
/write-chapter 1
```

系统自动执行：
1. WorldBuilder 查询章节涉及的世界观设定
2. CharacterKeeper 查询出场人物状态
3. ChapterWriter 起草正文
4. StyleGuard 文风报检
5. PolishEditor 深度质检
6. Coordinator 仲裁通过 → 归档

## 常用指令

| 指令 | 说明 |
|------|------|
| `/init` | 初始化系统（首次必做） |
| `/write-chapter N` | 写第 N 章 |
| `/status` | 查看项目进度 |
| `/patch-review` | 查看质检进化补丁 |
| `/world-status` | 查看世界观完整性 |

## 快捷模式

如果想跳过设定直接体验：

```
/write-chapter 1 --quickstart
```

系统会自动填充最小世界观（一个宗门、一个主角、一个敌人）。
注意：这是快速体验模式，后续必须补充世界观才能保持一致性。
